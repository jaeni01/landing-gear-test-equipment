# 랜딩기어 및 신뢰도 테스트 장비 (PLC Landing Gear Test Equipment)
> PLC와 공압 시스템으로 항공기 랜딩기어의 인입·전개 동작을 모사하고, 반복 사이클 신뢰도 테스트로 구동 내구성과 동작 안정성을 검증한 장비. 본인은 기구 설계·제작과 스프링 방식 충격흡수 구조, 실린더 연결 링크를 담당했습니다.

## 🎯 프로젝트 개요
항공기 랜딩기어의 인입(이륙 시)·전개(착륙 시) 동작을 PLC 시퀀스 제어와 공압 실린더로 자동화한 테스트 장비입니다. 실제 비행기 랜딩기어가 문이 열리며 실린더로 전개되고, 착륙 시 충격을 흡수하는 동작 메커니즘을 축소 모델로 구현했습니다. 충격흡수장치는 고무·공기 압축·오레오 등 여러 방식 중 **스프링 방식**으로 구현했습니다.

본인은 2인 팀에서 **기구 설계 및 제작**을 담당하여 랜딩기어 구조물과 실린더 연결 링크를 3D 프린팅으로 제작하고, 스프링을 이용한 충격흡수 구조를 설계했습니다. 단순 구동에 그치지 않고 리미트 스위치 기반 위치 감지, 반복 동작 카운트, 타이머를 이용한 고장 감지까지 포함해 실제 자동화 장비와 유사한 제어 구조를 구현했습니다.

## ✨ 주요 기능 / 담당 업무
> 본인 담당(기구 설계·제작) 항목은 **굵게** 표시했습니다.

- **랜딩기어 기구 설계 및 제작**: 비행기 랜딩기어의 인입·전개 동작을 모사하는 구조물을 설계하고, 3D 프린팅 기반으로 랜딩기어 구조물 및 실린더 연결 링크를 제작했습니다.
- **스프링 방식 충격흡수 구조 설계**: 착륙 시 충격을 흡수하는 장치를 고무·공기 압축·오레오 등 여러 방식을 검토한 뒤 스프링 방식으로 설계·구현했습니다.
- **실린더 연동 기구부 구성**: 공압 실린더가 랜딩기어를 전진·후진시키도록 실린더와 구조물을 연결하는 링크 기구를 설계했습니다.
- 랜딩기어 인입·전개 자동 제어 및 위치 감지 (리미트 스위치 기반) — *PLC 프로그래밍 담당: 정한길*
- 동작 횟수 카운트 및 타이머 기반 고장 감지 — *PLC 프로그래밍 담당: 정한길*

- 공압 실린더 / 솔레노이드 밸브

- 표시 램프(초록/빨강), 부저, 카운터

## 🔧 기술적 도전과 해결 (Technical Challenges)

### Q1. 여러 충격흡수 방식 중 무엇을 선택해 어떻게 구현했는가?
> **Challenge:** 실제 항공기 랜딩기어의 충격흡수장치는 고무, 스프링, 공기 압축, 오레오(oleo) 등 여러 방식이 존재한다. 축소 테스트 장비에서 착륙 충격을 흡수하는 구조를 직접 설계·제작해야 했다.
> **Solution:** 축소 모델에서 제작 가능성과 동작 재현성을 고려해 **스프링 방식**을 선택했다. 스프링을 이용한 충격흡수 구조를 설계하고 랜딩기어 구조물에 적용해, 전개 후 지면에 닿을 때의 충격을 흡수하도록 구현했다.

### Q2. 실린더 구동력을 랜딩기어 동작으로 전달하는 기구를 어떻게 구성했는가?
> **Challenge:** 공압 실린더의 직선 전진·후진 동작을 랜딩기어의 인입·전개 동작으로 변환·전달해야 했고, 전진/후진 위치에서 리미트 스위치가 정확히 감지되도록 기구 정밀도를 확보해야 했다.
> **Solution:** 실린더와 랜딩기어 구조물을 잇는 연결 링크 기구를 설계하고, 랜딩기어 구조물 및 링크를 3D 프린팅으로 제작했다. 실린더 전진 시 전개, 후진 시 인입이 이루어지고 각 끝단에서 리미트 스위치가 위치를 감지하도록 기구를 구성했다.

### Q3. 반복 동작에 견디는 테스트 장비 구조를 어떻게 만들었는가?
> **Challenge:** 본 장비는 반복 동작 횟수를 카운트하는 신뢰도 테스트 장비로, 수백 회 이상 반복되는 전개·인입 동작에서 구조물과 링크가 변형·파손되지 않아야 했다.
> **Solution:** 3D 프린팅 부품과 실린더 연결 링크를 반복 동작에 견디도록 설계·제작하고, 전진/후진 위치에서 리미트 스위치가 안정적으로 감지되도록 기구부를 조립했다. 이를 통해 반복 사이클 테스트 중 구동부의 내구성과 동작 안정성을 확인할 수 있도록 했다.
